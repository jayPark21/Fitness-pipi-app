## 작전 1 · 😴 피피 기분 자동변화 시스템

### 목적
리텐션 핵심 기능. 방치 시간에 따라 피피의 기분이 실시간으로 변화하여 사용자를 재방문하게 유도.

### 기분 변화 로직

```
lastInteractionTime 기준 경과 시간
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  0 ~  3시간  →  happy    🐧  "지금 딱 운동각인데..? 💪"
  3 ~  8시간  →  sad      😔  "나 요즘 좀 외로웠는데..."
  8 ~ 24시간  →  hungry   🍽️  "에너지 바닥났어~ 충전 필요함"
  24시간 이상 →  sleeping 💤  "(벌크업 중... 방해 금지 🛑)"
```

### 기술 구현

**`src/store/useStore.ts` 변경:**
- `checkAndUpdateMood()` 액션 추가
- `lastInteractionTime` 기준 경과 시간(ms) 계산
- mood가 실제 변화할 때만 `set()` 호출 → 불필요한 리렌더 방지
- mood 변경 후 0.5초 딜레이로 Firestore sync

```typescript
checkAndUpdateMood: () => {
    const prev = useStore.getState();
    const hoursElapsed = (Date.now() - new Date(prev.penguin.lastInteractionTime).getTime()) / 3600000;
    
    let newMood: PenguinState['mood'];
    if (hoursElapsed < 3)  newMood = 'happy';
    else if (hoursElapsed < 8)  newMood = 'sad';
    else if (hoursElapsed < 24) newMood = 'hungry';
    else                         newMood = 'sleeping';
    
    if (newMood !== prev.penguin.mood) {
        set({ penguin: { ...prev.penguin, mood: newMood } });
        setTimeout(() => useStore.getState().syncWithFirestore(), 500);
    }
},
```

**`src/pages/Dashboard.tsx` 변경:**
- `checkAndUpdateMood` 디스트럭처링 추가
- 앱 열 때 즉시 + 5분마다 주기적으로 체크하는 `useEffect` 추가

```typescript
useEffect(() => {
    checkAndUpdateMood(); // 앱 열 때 즉시
    const interval = setInterval(checkAndUpdateMood, 5 * 60 * 1000); // 5분마다
    return () => clearInterval(interval);
}, [checkAndUpdateMood]);
```

### 리텐션 효과
- 3시간 이상 미접속 → 피피가 외로워짐 → 알림과 결합 시 재방문율 상승 기대
- 기분에 따라 말풍선 멘트도 달라짐 (기존 구현과 연계)

## 작전 2 · 🎉 레벨업 폭죽 연출

### 목적
XP 누적으로 레벨업 달성 시 화려한 시각적 피드백으로 성취감 극대화.

### 연출 흐름
```
피피 터치 → XP 획득 → 레벨 달성!
           ↓
  🎉 60개 컬러 콘페티 화면 중앙 폭발!
           ↓
  ┌──────────────────────┐
  │ 🎉                   │
  │   LEVEL UP!          │
  │      ★ 3 ★           │  ← 스프링 애니메이션
  │  피피가 성장했어! 🐧✨ │
  │ 🎊                   │
  └──────────────────────┘
           ↓
     3초 후 자동 사라짐
```

### 기술 구현

**`src/store/useStore.ts` 변경:**

1. `PenguinState` 인터페이스에 `justLeveledUp: boolean` 필드 추가
2. `clearLevelUp()` 액션 추가 — 애니메이션 종료 후 플래그 초기화
3. `interactWithPipi()` 내 레벨업 감지 로직:

```typescript
let didLevelUp = false;
if (newXp >= nextXp) {
    level += 1;
    nextXp = Math.floor(nextXp * 1.5);
    didLevelUp = true; // 🎉 레벨업!
}
// ...
justLeveledUp: didLevelUp, // 📡 Dashboard에 신호 전달
```

**`src/pages/Dashboard.tsx` 변경:**

- `confetti`, `showLevelUp`, `levelUpNum` 상태 추가
- `penguin.justLeveledUp` 감지 `useEffect`:

```typescript
useEffect(() => {
    if (!penguin.justLeveledUp) return;
    
    const COLORS = ['#FF6B6B', '#FFD93D', '#6BCB77', '#4D96FF', 
                    '#FF6FC8', '#C77DFF', '#FF9A3C', '#00F5D4'];
    // 60개 파티클 생성 (360° 랜덤 방향)
    const particles = Array.from({ length: 60 }, (_, i) => ({...}));
    setConfetti(particles);
    setShowLevelUp(true);
    setLevelUpNum(penguin.friendshipLevel);
    
    setTimeout(() => {
        setConfetti([]); setShowLevelUp(false); clearLevelUp();
    }, 3000);
}, [penguin.justLeveledUp]);
```

## 작전 4 · 📺 운동 화면(Workout) UI/UX 대개조

### 목적
사용자의 운동 편의성을 극대화하기 위해 영상 짤림 해결, 준비 시간 도입, 자세 안내 이미지 추가.

### 주요 변경 사항
1. **운동 준비 시간(Prep Phase) 5초 도입**: 각 운동 세트 시작 전 5초간 "MISSION PREPARE" 단계를 두어 사용자가 자세를 잡을 여유를 제공.
2. **준비 자세 이미지(Pose Image) 가이드**: 준비 시간 동안 해당 운동의 바른 자세 이미지를 Pexels Photo API를 통해 실시간으로 호출, 오버레이로 크게 표시하여 초보자도 쉽게 따라 할 수 있게 함.
3. **영상 출력 최적화 (`object-contain`)**: 기존 `object-cover` 방식에서 `object-contain`으로 변경하여, 어떤 비율의 영상이라도 인물 전신이 짤림 없이 화면에 꽉 차게 표시.
4. **시청각 피드백 강화**:
    - **오디오**: 준비 카운트다운(440Hz)과 시작 알림(1320Hz)의 주파수를 다르게 설정하여 직관적인 피드백 제공.
    - **비주얼**: 준비 단계에서 배경 영상을 블러(Blur) 처리하고 텍스트와 준비 이미지를 강조하여 집중도 향상.
5. **마운틴 클라이머 검색 쿼리 정밀화**: `"mountain climber floor exercise ab"` 쿼리를 적용하여 엉뚱한 등산 영상 대신 정확한 복근 운동 영상이 나오도록 교정.
