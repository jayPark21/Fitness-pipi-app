## 🚀 오늘의 주요 작전 (레트로 오프닝 감성 강화)

### 1. 👾 8비트 레트로 인트로 화면 (Intro Screen) 구축
- **배경**: 대표님께서 직접 제작하신 8비트 레트로풍 BGM을 앱의 첫인상으로 활용하기 위해 전용 스타트 화면 기획.
- **기술 구현**:
    - **`Intro.tsx` 생성**: 픽셀 아트 감성의 UI 디자인 (Google Font 'Press Start 2P' 적용).
    - **CRT 효과**: 옛날 브라운관 TV 느낌의 스캔라인(Scanline) 및 그라데이션 오버레이 적용.
    - **인터랙션 연동**: 브라우저 오디오 정책을 준수하기 위해 "PRESS START" 클릭 시 BGM이 재생되도록 구현.
    - **애니메이션**: `framer-motion`을 사용하여 타이틀과 펭귄 아이콘에 역동적인 움직임 부여.
- **라우팅 조정**: 앱의 첫 진입점(`/`)을 인트로 화면으로 설정하고, 기존 온보딩은 `/onboarding`으로 이동.

### 2. 🎵 인트로 BGM 에셋 최종 연동 완료
- **작업 내용**: 대표님께서 삽입하신 `intro_bgm.mp3` 파일을 `Intro.tsx`에 정적 임포트(`Static Import`) 방식으로 연동.
- **효과**: 빌드 시 파일 해싱 및 최적화가 적용되어 경로 오류 없이 안정적으로 재생 가능.
